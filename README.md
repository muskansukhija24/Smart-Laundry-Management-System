# Smart Laundry Management System

## 📌 Overview
This project is a Java-based application to automate laundry services.

## 🚀 Features
- User Registration
- Laundry Booking
- Order Tracking
- Billing System

## 🛠️ Tech Stack
- Java
- File Handling / MySQL

## ▶️ How to Run
Run Main.java
